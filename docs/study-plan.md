## 🔥 Priority List (nếu có ít thời gian hơn)

Nếu chỉ có 2 tuần, ưu tiên theo thứ tự:

1. **Algorithms**: làm ít nhất 20 bài, focus pattern Two Pointers + Hash Map + Trees + Sliding Window
2. **Backend**: Databases + Networking + Concurrency + API Design (4 file)
3. **System Design**: học framework + 3 bài mẫu ✅ + tự luyện 1 bài
4. **Frontend**: React Core + State Management + TypeScript (3 file)
5. **Behavioral**: 3 câu chuyện STAR
6. **DevOps**: Docker + CI/CD (2 file)

## 🎙️ Mock Interview Routine

Khi tự luyện system design hoặc behavioral, làm theo đúng nhịp thật:

1. Đặt timer đúng thời gian (45 phút system design, 5 phút behavioral).
2. **Đứng lên** và nói to (hoặc record lại).
3. Xem lại recording → tự đánh giá.
4. Đối chiếu với docs.
